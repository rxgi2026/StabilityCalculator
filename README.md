# Calculadora SDS IGF-1 — versão final

## Como abrir

1. Descompactar a pasta ZIP.
2. Abrir o ficheiro `index.html` num browser moderno.
3. Preencher a calculadora, clicar em **Calcular Score SDS** e depois em **Descarregar PDF**.

## Notas

- Esta versão inclui o logótipo da Novo Nordisk no header do PDF.
- O gráfico do PDF foi ajustado para manter a proporção correta e ficar mais pequeno.
- O PDF inclui aviso final de confidencialidade e privacidade em inglês e português.
- A página utiliza Chart.js, jsPDF e html2pdf por CDN; por isso, para gerar gráfico/PDF, é recomendável abrir com ligação à internet.
- Antes de publicação, validar referências, algoritmos, dados de referência, privacidade, farmacovigilância e aprovação MLR local.
